# Fahrschul Classroom

Live-Classroom-System für Fahrschul-Theorie-Unterricht.

## Services

- `frontend/` — React PWA (Vite + TypeScript)
- `backend/` — Node.js + Express + Socket.io

## Setup

Siehe `docs/plans/` für Implementierungsplan.

## Development

```bash
# Backend starten (Port 3000)
cd backend
npm run dev

# Frontend starten (Port 5173)
cd frontend
npm run dev
```

## Tech-Stack

| Bereich | Technologie |
|---|---|
| Frontend | React 19 + Vite + TypeScript |
| Backend | Node.js + Express + Socket.io |
| Datenbank | Supabase (PostgreSQL) |
| Echtzeit | Socket.io |
| Video | Jitsi Meet |
